                    ┌────────────────┼────────────────────┐

            ┌───────────────┐ ┌──────────────┐ ┌──────────────────┐

            └───────┬───────┘ └──────┬───────┘ └────────┬─────────┘

                    ▼                ▼                   ▼

## 2. 学习路径流

```
    ユーザー
       │
       ▼
  ┌──────────────────┐
  │ 浏览学习路径       │
  │ (3条角色化路径)    │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐     ┌──────────────────┐
  │ 选择一条路径      │────▶│ 查看路径详情      │
  └──────────────────┘     │ (步骤 / 时间)     │
                           └────────┬─────────┘
                                    │
                                    ▼
                           ┌──────────────────┐
                           │ 开始学习          │
                           │ (按顺序看文章)     │
                           └────────┬─────────┘
                                    │
                                    ▼
                           ┌──────────────────┐
                           │ 完成所有步骤      │
                           │ → 获得修了证书    │
                           └──────────────────┘
```

## 3. 每日一题流

```
    ユーザー
       │
       ▼
  ┌──────────────────┐
  │ 看到 Daily Quiz   │
  │ (首页/固定入口)    │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐
  │ 阅读问题 & 选项   │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐
  │ 点击选项          │
  └────────┬─────────┘
           │
    ┌──────┴──────┐
    ▼              ▼
  ┌──────┐     ┌──────────┐
  │ 正解! │     │ おしい!   │
  │ ✓ 绿  │     │ ✕ 红色   │
  └──┬───┘     └────┬─────┘
     │              │
     └──────┬───────┘
            ▼
  ┌──────────────────┐
  │ 显示熊猫解説      │
  │ (解析 bubble)     │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐
  │ 点击「次の問題」   │
  │ → 循环下一题      │
  └──────────────────┘
```

## 4. 案件探索流

```
    ユーザー (访客/注册)
       │
       ▼
  ┌──────────────────────┐
  │ 案件跑马灯（首页顶部） │
  │ 或 案件専用セクション  │
  └──────────┬───────────┘
             │
             ▼
  ┌──────────────────────┐
  │ 选择技能模块 (匹配)   │
  │ FI / CO / MM / ...   │
  └──────────┬───────────┘
             │
             ▼
  ┌──────────────────────┐
  │ 浏览案件卡片          │
  │ (匹配排序优先)        │
  └──────────┬───────────┘
             │
        ┌────┴────┐
        ▼         ▼
  ┌──────────┐ ┌──────────┐
  │ 点击卡片   │ │ 继续浏览  │
  └────┬─────┘ └──────────┘
       │
       ▼
  ┌──────────────────────┐
  │ 案件详情 (Modal)      │
  │ 薪资/期間/場所/スキル │
  └──────────┬───────────┘
             │
       ┌─────┴─────┐
       ▼           ▼
  ┌──────────┐ ┌──────────┐
  │ 応募する   │ │ 一覧に戻る │
  └────┬─────┘ └──────────┘
       │
       ▼
  ┌──────────────────────┐
  │ 応募フォーム          │
  │ 名前/連絡先/履歴書    │
  └──────────┬───────────┘
             │
             ▼
  ┌──────────────────────┐
  │ 応募完了!             │
  │ Thank you / 確認      │
  └──────────────────────┘
```

## 5. 用户注册流

```
    ユーザー
       │
       ▼
  ┌──────────────────┐
  │ 点击「無料登録」   │
  │ (CTA / 导航按钮)   │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐
  │ 注册表单          │
  │ メール / PW / 名前 │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐
  │ 确认邮件          │
  │ (メール認証)       │
  └────────┬─────────┘
           │
           ▼
  ┌──────────────────┐
  │ 注册完成          │
  │ → マイページ      │
  └──────────────────┘
```

## 6. 文章互动流

```
    ユーザー (阅读文章时)
       │
       ├── ▶ 阅读 TOC → 跳转章节
       ├── ▶ 点击 Reaction (👍❤🎋🙏)
       ├── ▶ 点击 SNS 分享
       ├── ▶ 点击「保存」(收藏)
       ├── ▶ 阅读相关文章
       └── ▶ 浮动熊猫助手
              │
              ▼
         ┌──────────────────┐
         │ 熊猫助手弹出气泡   │
         │ + 关键词 Tag      │
         └──────────────────┘
```
